# React_Tutorial_Part1

## React를 실습하기 위한 환경만들기
  1. node.js 설치
  - node.js 다운로드 ( https://nodejs.org/en/download/?utm_source=blog ) 사이트가기
  - Mac 또는 Windows 선택후 다운로드
  - 터미널창을 열고 ``` > node -v ``` v18.15.0 이런식으로 숫자가 나오면 설치가 된것입니다  
  2. 에디터 설치 
  - vs code 를 설치합니다.
  - 확장프로그램을 설치합니다
